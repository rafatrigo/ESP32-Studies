## üìù How to Run These Projects

Since this repository uses **ESP-IDF**, follow these steps for any of the project folders:

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/rafatrigo/ESP32-Studies.git
    cd ESP32-Studies
    ```

2.  **Navigate to a specific project:**
    ```bash
    cd wifi-connection
    ```

3.  **Read the project README.md**

In the project README probably you will find more details about how to run the project.
